Git 版本控制

date: 2017/05/18

Download Git

資料夾初始化，開始進行git版本控制

cmd》git init

版本控制─本機

  1. cmd》git add .

    2. //上傳到暫存區(staging area)

  2. cmd》git commit -m "version X"

    3. //上傳到本機資料庫 (repository)

版本控制─遠端

  1. 到Github.com上複製遠端儲存庫網址

  2. cmd》git remote add origin http://~.git

    3. //連結本機資料庫和遠端資料庫(資料庫名稱設一樣的)

  3. cmd》git push origin master

    4. //把本機資料庫的檔案上傳到遠端資料庫

把遠端資料庫下載到本機

  1. 在Github.com上複製遠端儲存庫網址(clone)

  2. cmd》git clone "http://~.git"

    3. //把遠端檔案下載到本機
